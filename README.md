# Feasto

Feasto is a modern, premium food delivery frontend web application inspired by leading platforms like Swiggy and Zomato. It is built with a responsive, high-performance tech stack and features a clean, attractive user interface with dynamic animations, realistic mockup data, and a complete dark mode implementation.

## Tech Stack

This project is built using the following technologies:
- **React 19** - UI Library
- **Vite** - Next Generation Frontend Tooling
- **Tailwind CSS** - Utility-first styling framework
- **Redux Toolkit** - State management
- **React Router** - Client-side routing
- **Framer Motion** - Fluid animations and transitions
- **React Icons & Lucide React** - High quality SVG icons

## Setup Instructions

To get this project running locally on your machine, follow these steps:

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/yourusername/feasto-frontend.git
   cd feasto-frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

Once the development server is running, the application will be accessible at http://localhost:5173.

## Project Structure
- `src/components`: Reusable UI elements (Navbar, Footer, Cards, etc.)
- `src/pages`: Distinct application views (Home, Splash, Auth, Cart, etc.)
- `src/redux`: Redux store configuration and slices
- `src/index.css`: Tailwind configuration and base styles
